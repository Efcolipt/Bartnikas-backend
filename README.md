## Installation

```bash
$ npm install

$ cp .env.example .env
# заполнить .env
# создать учетку админа
# запустить сиды

# возможно нужно будет расширить права на папку upload, чтобы файлы из этой папки были доступны в инете
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
